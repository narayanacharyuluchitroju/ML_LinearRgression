In order to execute the below .ipynb(Q1_LinearRegression_1variable,Q2_LinearRegression_2Variables,Q3_stepwiseBackwardRegression,Q3_stepwiseForwardRegression,Q4_Regularization_FeatureScalling) files below files are required:
LinearRegression.py (contains custom Linear regression classes with one variable,two variables, three variables, four variables and five variables)
LinearRegression_Regularized.py (added regularization to the cost function and its respective derivatives)
cancer_dataset_wpbc.csv

All the above mentioned files must lie in same directory.
